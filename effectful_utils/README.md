# effectful_utils

Dart utility extensions for strings, collections, numbers, time, ranges, comparators, and function composition.

## Installation

```yaml
dependencies:
  effectful_utils:
    git:
      url: https://github.com/editorstack/effectful
      path: effectful_utils
```

## Usage

```dart
import 'package:effectful_utils/effectful_utils.dart';
```

See [`lib/effectful_utils.dart`](./lib/effectful_utils.dart) for the full export list.
